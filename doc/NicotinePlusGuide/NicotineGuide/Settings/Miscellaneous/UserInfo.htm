2.4.2 Nicotine+ Settings: User info


When users request your User info, they get to see some info provided by you. You can provide that kind of info here.

Self description

Put here what you want others to know.

You can only add text. HTML tags won't work, instead you see the syntax.

If you have a webpage or profile somewhere, add it here, instead of having a long self description

Image

picture of yourself, pet, car, house, whatever you want others to see

Don't choose a high, big resolution picture. It takes time to load and you experice a slow Nicotine+ if everytime your User info is requested.
