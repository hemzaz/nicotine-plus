2.2.1 Nicotine+ Settings: Server


Server settings allows you to choose the server you wish to connect to and configure your username, password and connection ports.

Server

allows you to choose which server you want connect to
  • default is: server.slsknet.net:2242
  • unless you know what you doing, leave this option default!

Login

field to fill in your username
  • if it's the first time you're connecting to Soulseek Network, it can happen that your choosen username is already in use
  • Look in Log window to see if your username is in use. If it's in use you see: INVALID PASS
  • If it's not in use, well, you just connect and enjoy Nicotine+, Soulseek Network and the whole experince :)

Password

field for your password

Choose proper password. You can't recover password the easy way!


Network Character Encoding

allows you to choose your keyboard layout
  • default is utf-8
  • unless you have some keyboard from stone-age, there's no need for changing this

Client connection ports

ports needed by Nicotine+ to function well.
  • default is 2234 - 2239
  • If ports are open and can recieve direct connection, check that box

CTCP VERSION

users are allowed to check which version Nicotine+ you have.
  • default is checked
  • If other users have this option checked, you can check their version.
  • You can do this in Private chat session
