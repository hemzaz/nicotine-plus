2.2.4 Nicotine+ Settings: Geo Block


Geo Block is NOT available for Windows versions of Nicotine+

Geo Block allows you to block specific users from a country.

Enable geographical blocker

When checked you can configure 2 options as explained below.

Geographica paranoia

Blocks users who possibly use proxy server to hide their real IP

Country codes


Guides explaination regarding using country codes needs some research! Information showed might be wrong!

Use this option to block users from specific countries by typing country codes seperated by comma (,)
example: us, nl, pl, se, de
