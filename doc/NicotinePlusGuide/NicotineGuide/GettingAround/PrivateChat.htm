3.3 Nicotine+ Getting Around: Private chat

Private chat window consist of a few parts.

1. username tabs

  • Every user you having private chat with has a tab. When you chat with more than one user you can switch between them by clicking on the tab

2. Private chat itself

  • That's the Private chat itself. Your lines are colored in blue and the other persons coversation is black
  • Every message is shown this way: day month date hour minutes sec year [username] message (example: Thu Jun 15 17:13:46 2006 [user_name])

3. Message box

  • There you can typ whatever you want to say and hit enter

4. Log

  • Saves the conversation

5. Close

  • Closes chat session
